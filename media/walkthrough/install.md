## DebugPilot Installed

DebugPilot is an MCP server that exposes VS Code's Debug Adapter Protocol to AI agents.

Once activated, it starts an HTTP+SSE MCP server on a local port. Look for the **DebugPilot :PORT** indicator in the status bar to confirm it's running.

### What you get

- A local MCP server your AI agent can connect to
- Full debugger control: breakpoints, stepping, variable inspection
- Works with any VS Code debugger (Node.js, Python, C++, Go, etc.)
